24.1 为什么 Agent 可观测性是「必需品」而非「可选项」？

 

传统 API 监控：

请求 → 响应 → 记录 HTTP 状态码 + 延迟 → 完成

 

Agent 监控的特殊性：

一次「请求」包含 N 次 LLM 调用 + M 次工具调用

错误可能在第 15 步才出现（难以定位）

Multi-Agent 的调用链复杂（Agent A → Agent B → Agent C → ...）

非确定性输出（同一输入，两次结果可能不同）

 

Agent 可观测性的 4 大支柱：

 

  ┌──────────────┬──────────────────────────────────────┐
  │    支柱        │              需要回答的问题            │
  ├──────────────┼──────────────────────────────────────┤
  │ 1. Tracing   │ Agent 执行了哪些步骤？每个步骤耗时？    │
  │ 2. Metrics   │ 整体成功率？P50/P99 延迟？Token消耗？   │
  │ 3. Evaluation │ 输出质量如何？是否满足用户需求？        │
  │ 4. Alerts    │ 什么情况需要人工介入？                  │
  └──────────────┴──────────────────────────────────────┘

 

 

24.2 Tracing —— 看见 Agent 的「每一根神经」

 

Tracing 的层次模型：

 

Level 1: 单次 LLM 调用 Trace

记录: model / input_tokens / output_tokens / latency / status

 

Level 2: Chain/Agent 级别 Trace

记录: 完整的 Agent 执行链
- 每个 tool_call: tool_name + args + result + latency
- 每个 LLM 调用: system_prompt_snapshot + user_message
- 每个步骤间的关系（父子 span）

 

Level 3: Multi-Agent Trace

记录: Agent 之间的消息传递
- 哪个 Agent 把任务委派给了哪个 Agent
- 委派的 context 是什么
- 返回的 artifact 是什么

 

Trace 的核心数据结构：Span

 

  ┌──────────────────────────────────────────────┐
  │                  Trace (根)                    │
  │  ├── Span: Agent 步骤 1 (thinking)            │
  │  │   ├── Span: LLM 调用                       │
  │  │   │   ├── model: gpt-4o-mini              │
  │  │   │   ├── input_tokens: 1250              │
  │  │   │   ├── output_tokens: 86               │
  │  │   │   └── latency_ms: 1520                │
  │  │   └── Span: 工具调用                        │
  │  │       ├── tool: search                    │
  │  │       ├── args: {"query": "AI Agent"}      │
  │  │       ├── result_summary: "找到5条结果"     │
  │  │       └── latency_ms: 350                 │
  │  ├── Span: Agent 步骤 2 (tool_call)           │
  │  └── Span: Agent 步骤 3 (done)                │
  └──────────────────────────────────────────────┘

import time
import json
import hashlib
from typing import Optional
from dataclasses import dataclass, field
from collections import defaultdict
from contextlib import contextmanager


@dataclass
class TraceSpan:
    class=class="s">"s">class="s">""class="s">"Tracing Span —— 可观测性数据的基本单元。"class="s">""
    trace_id: str
    span_id: str
    parent_span_id: Optional[str]
    name: str                    class=class="s">"h"># class="s">"llm_call" / class="s">"tool_call" / class="s">"agent_step"
    start_time: float
    end_time: Optional[float] = None
    attributes: dict = field(default_factory=dict)
    status: str = class="s">"ok"           class=class="s">"h"># ok / error
    children: list = field(default_factory=list)

    @property
    def latency_ms(self) -> float:
        if self.end_time is None:
            return 0
        return (self.end_time - self.start_time) * 1000


class AgentTracer:
    class=class="s">"s">class="s">"""Agent 专用 Tracer —— 模拟 LangSmith/LangFuse 的功能。

    实现：
      1. Trace 管理（创建/结束/嵌套）
      2. LLM 调用自动记录
      3. 工具调用自动记录
      4. 跨 Agent 调用链
    class="s">"""

    def __init__(self, project_name: str = class="s">"agent-study"):
        self.project_name = project_name
        self.traces: dict[str, list[TraceSpan]] = defaultdict(list)
        self._active_spans: dict[str, TraceSpan] = {}
        self.metrics = defaultdict(int)

    def start_trace(self, name: str,
                    session_id: str = None) -> str:
        class=class="s">"s">class="s">"""开始一个新的 Trace。

        Args:
            name: Trace 名称（如 class="s">"user_query"）。
            session_id: 会话 ID。

        Returns:
            trace_id。
        class="s">"""
        trace_id = session_id or hashlib.md5(
            fclass="s">"{name}-{time.time()}".encode()
        ).hexdigest()[:16]

        span = TraceSpan(
            trace_id=trace_id,
            span_id=fclass="s">"span_{len(self.traces[trace_id])}",
            parent_span_id=None,
            name=name,
            start_time=time.time(),
        )
        self.traces[trace_id].append(span)
        self._active_spans[trace_id] = span
        return trace_id

    @contextmanager
    def span(self, trace_id: str, name: str, **attributes):
        class=class="s">"s">class="s">"""创建一个子 Span（上下文管理器）。

        用法:
          with tracer.span(tid, class="s">"llm_call", model=class="s">"gpt-4o"):
              result = llm.invoke(...)

        Args:
            trace_id: 父 Trace ID。
            name: Span 名称。
            **attributes: 附加属性。
        class="s">"""
        parent = self._active_spans.get(trace_id)

        span = TraceSpan(
            trace_id=trace_id,
            span_id=fclass="s">"span_{len(self.traces[trace_id])}",
            parent_span_id=parent.span_id if parent else None,
            name=name,
            start_time=time.time(),
            attributes=attributes,
        )
        self.traces[trace_id].append(span)
        if parent:
            parent.children.append(span)

        try:
            yield span
        except Exception as e:
            span.status = class="s">"error"
            span.attributes[class="s">"error"] = str(e)
            raise
        finally:
            span.end_time = time.time()
            span.attributes[class="s">"latency_ms"] = span.latency_ms

    def log_llm_call(self, trace_id: str, model: str,
                     input_tokens: int, output_tokens: int,
                     latency_ms: float, success: bool = True):
        class=class="s">"s">class="s">""class="s">"记录一次 LLM 调用（便捷方法）。"class="s">""
        with self.span(trace_id, class="s">"llm_call",
                       model=model,
                       input_tokens=input_tokens,
                       output_tokens=output_tokens,
                       success=success) as span:
            span.end_time = span.start_time + latency_ms / 1000

        self.metrics[class="s">"total_llm_calls"] += 1
        self.metrics[class="s">"total_input_tokens"] += input_tokens
        self.metrics[class="s">"total_output_tokens"] += output_tokens
        if not success:
            self.metrics[class="s">"failed_llm_calls"] += 1

    def log_tool_call(self, trace_id: str, tool_name: str,
                      input_args: dict, output_summary: str,
                      latency_ms: float, success: bool = True):
        class=class="s">"s">class="s">""class="s">"记录一次工具调用（便捷方法）。"class="s">""
        with self.span(trace_id, class="s">"tool_call",
                       tool_name=tool_name,
                       input_args=str(input_args)[:200],
                       output_summary=output_summary[:200],
                       success=success) as span:
            span.end_time = span.start_time + latency_ms / 1000

        self.metrics[class="s">"total_tool_calls"] += 1
        if not success:
            self.metrics[class="s">"failed_tool_calls"] += 1

    def get_trace(self, trace_id: str) -> Optional[dict]:
        class=class="s">"s">class="s">""class="s">"获取 Trace 的完整信息（用于调试和展示）。"class="s">""
        spans = self.traces.get(trace_id, [])
        if not spans:
            return None

        total_latency = sum(
            s.latency_ms for s in spans
            if s.end_time is not None
        )
        failed = sum(1 for s in spans if s.status == class="s">"error")

        return {
            class="s">"trace_id": trace_id,
            class="s">"span_count": len(spans),
            class="s">"total_latency_ms": round(total_latency, 1),
            class="s">"failed_spans": failed,
            class="s">"spans": [
                {
                    class="s">"name": s.name,
                    class="s">"latency_ms": round(s.latency_ms, 1),
                    class="s">"status": s.status,
                    class="s">"attributes": {
                        k: v for k, v in s.attributes.items()
                        if k in (class="s">"model", class="s">"tool_name", class="s">"input_tokens",
                                 class="s">"output_tokens", class="s">"success")
                    },
                    class="s">"children": len(s.children),
                }
                for s in spans
            ],
        }

    def get_metrics(self) -> dict:
        class=class="s">"s">class="s">""class="s">"获取聚合指标。"class="s">""
        return {
            class="s">"project": self.project_name,
            class="s">"traces": len(self.traces),
            class="s">"metrics": dict(self.metrics),
            class="s">"avg_tokens_per_call": (
                self.metrics[class="s">"total_input_tokens"] /
                max(self.metrics[class="s">"total_llm_calls"], 1)
            ),
            class="s">"tool_success_rate": (
                1 - self.metrics[class="s">"failed_tool_calls"] /
                max(self.metrics[class="s">"total_tool_calls"], 1)
            ) if self.metrics[class="s">"total_tool_calls"] > 0 else 0,
        }

    def print_trace_tree(self, trace_id: str, indent: int = 0):
        class=class="s">"s">class="s">""class="s">"打印 Trace 树形结构（直观展示执行流程）。"class="s">""
        spans = self.traces.get(trace_id, [])
        if not spans:
            print(class="s">"  无 Trace 数据")
            return

        class=class="s">"h"># 找到根 Span
        root = None
        for s in spans:
            if s.parent_span_id is None:
                root = s
                break

        if root is None:
            root = spans[0]

        def _print_span(span: TraceSpan, level: int):
            prefix = class="s">"  " * level + (class="s">"├─ " if level > 0 else class="s">"")
            status_icon = class="s">"✅" if span.status == class="s">"ok" else class="s">"❌"
            attrs = class="s">""
            if class="s">"model" in span.attributes:
                attrs += fclass="s">" [{span.attributes[class='s'>'model']}]"
            if class="s">"tool_name" in span.attributes:
                attrs += fclass="s">" [{span.attributes[class='s'>'tool_name']}]"
            print(fclass="s">"{prefix}{status_icon} {span.name}{attrs} "
                  fclass="s">"({span.latency_ms:.0f}ms)")

            for child in span.children:
                _print_span(child, level + 1)

        _print_span(root, 0)

    def print_dashboard(self):
        class=class="s">"s">class="s">""class="s">"打印监控仪表盘（模拟 Grafana 面板）。"class="s">""
        m = self.get_metrics()
        print(class="s">"=" * 55)
        print(class="s">"  📊 Agent 监控仪表盘")
        print(class="s">"=" * 55)
        print(fclass="s">"  项目: {m[class='s'>'project']}")
        print(fclass="s">"  Traces 总数: {m[class='s'>'traces']}")
        print(fclass="s">"  LLM 调用: {m[class='s'>'metrics'].get(class='s'>'total_llm_calls', 0)}")
        print(fclass="s">"  总输入 Tokens: {m[class='s'>'metrics'].get(class='s'>'total_input_tokens', 0):,}")
        print(fclass="s">"  总输出 Tokens: {m[class='s'>'metrics'].get(class='s'>'total_output_tokens', 0):,}")
        print(fclass="s">"  平均 Tokens/次: {m[class='s'>'avg_tokens_per_call']:.0f}")
        print(fclass="s">"  工具调用: {m[class='s'>'metrics'].get(class='s'>'total_tool_calls', 0)}")
        print(fclass="s">"  工具成功率: {m[class='s'>'tool_success_rate']:.0%}")
        status = class="s">"🟢 正常" if m[class='s'>'metrics'].get(class='s'>'failed_llm_calls', 0) == 0 else class="s">"🔴 异常"
        print(fclass="s">"  系统状态: {status}")
        print(class="s">"-" * 55)

24.3 LangSmith vs LangFuse —— 两大平台对比

 

  ┌──────────────┬─────────────────────┬─────────────────────┐
  │     维度      │      LangSmith       │      LangFuse        │
  ├──────────────┼─────────────────────┼─────────────────────┤
  │ 开发者        │ LangChain 公司       │ 独立开源社区         │
  │ 开源          │ 部分开源             │ 完全开源 (MIT)       │
  │ 部署          │ SaaS (托管)          │ SaaS + 自托管 (Docker/K8s)│
  │ LangChain集成 │ 原生（环境变量开箱）   │ 通过回调集成         │
  │ 定价          │ 免费层 + Pro/Enterprise│ 免费层 + Cloud/Enterprise│
  │ 特色功能       │ Hub (Prompt共享)     │ Prompt Management    │
  │ 评测          │ LLM-as-Judge 评测     │ Dataset Runs 评测    │
  └──────────────┴─────────────────────┴─────────────────────┘

 

选型建议：

  • 重度用 LangChain/LangGraph → LangSmith（原生集成）
  • 需要自托管 + 完全开源 → LangFuse
  • 小团队快速上手 → LangSmith Cloud
  • 需要 Prompt 版本管理 → LangFuse

☐ 连续 N 次相同错误 → 死循环告警

    ☐ 用户输入异常（Prompt Injection 检测触发）
☐ 输出异常（内容安全审核触发）
☐ 流量异常（Rate Limiting 触发）

24.1 为什么 Agent 可观测性是「必需品」而非「可选项」？
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

传统 API 监控：
  请求 → 响应 → 记录 HTTP 状态码 + 延迟 → 完成

Agent 监控的特殊性：
  一次「请求」包含 N 次 LLM 调用 + M 次工具调用
  错误可能在第 15 步才出现（难以定位）
  Multi-Agent 的调用链复杂（Agent A → Agent B → Agent C → ...）
  非确定性输出（同一输入，两次结果可能不同）

Agent 可观测性的 4 大支柱：

  ┌──────────────┬──────────────────────────────────────┐
  │    支柱        │              需要回答的问题            │
  ├──────────────┼──────────────────────────────────────┤
  │ 1. Tracing   │ Agent 执行了哪些步骤？每个步骤耗时？    │
  │ 2. Metrics   │ 整体成功率？P50/P99 延迟？Token消耗？   │
  │ 3. Evaluation │ 输出质量如何？是否满足用户需求？        │
  │ 4. Alerts    │ 什么情况需要人工介入？                  │
  └──────────────┴──────────────────────────────────────┘


24.2 Tracing —— 看见 Agent 的「每一根神经」
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Tracing 的层次模型：

  Level 1: 单次 LLM 调用 Trace
    记录: model / input_tokens / output_tokens / latency / status

  Level 2: Chain/Agent 级别 Trace
    记录: 完整的 Agent 执行链
    - 每个 tool_call: tool_name + args + result + latency
    - 每个 LLM 调用: system_prompt_snapshot + user_message
    - 每个步骤间的关系（父子 span）

  Level 3: Multi-Agent Trace
    记录: Agent 之间的消息传递
    - 哪个 Agent 把任务委派给了哪个 Agent
    - 委派的 context 是什么
    - 返回的 artifact 是什么

Trace 的核心数据结构：Span

  ┌──────────────────────────────────────────────┐
  │                  Trace (根)                    │
  │  ├── Span: Agent 步骤 1 (thinking)            │
  │  │   ├── Span: LLM 调用                       │
  │  │   │   ├── model: gpt-4o-mini              │
  │  │   │   ├── input_tokens: 1250              │
  │  │   │   ├── output_tokens: 86               │
  │  │   │   └── latency_ms: 1520                │
  │  │   └── Span: 工具调用                        │
  │  │       ├── tool: search                    │
  │  │       ├── args: {class="s">"query": class="s">"AI Agent"}      │
  │  │       ├── result_summary: class="s">"找到5条结果"     │
  │  │       └── latency_ms: 350                 │
  │  ├── Span: Agent 步骤 2 (tool_call)           │
  │  └── Span: Agent 步骤 3 (done)                │
  └──────────────────────────────────────────────┘

import time
import json
import hashlib
from typing import Optional
from dataclasses import dataclass, field
from collections import defaultdict
from contextlib import contextmanager


@dataclass
class TraceSpan:
    class=class="s">"s">class="s">""class="s">"Tracing Span —— 可观测性数据的基本单元。"class="s">""
    trace_id: str
    span_id: str
    parent_span_id: Optional[str]
    name: str                    class=class="s">"h"># class="s">"llm_call" / class="s">"tool_call" / class="s">"agent_step"
    start_time: float
    end_time: Optional[float] = None
    attributes: dict = field(default_factory=dict)
    status: str = class="s">"ok"           class=class="s">"h"># ok / error
    children: list = field(default_factory=list)

    @property
    def latency_ms(self) -> float:
        if self.end_time is None:
            return 0
        return (self.end_time - self.start_time) * 1000


class AgentTracer:
    class=class="s">"s">class="s">"""Agent 专用 Tracer —— 模拟 LangSmith/LangFuse 的功能。

    实现：
      1. Trace 管理（创建/结束/嵌套）
      2. LLM 调用自动记录
      3. 工具调用自动记录
      4. 跨 Agent 调用链
    class="s">"""

    def __init__(self, project_name: str = class="s">"agent-study"):
        self.project_name = project_name
        self.traces: dict[str, list[TraceSpan]] = defaultdict(list)
        self._active_spans: dict[str, TraceSpan] = {}
        self.metrics = defaultdict(int)

    def start_trace(self, name: str,
                    session_id: str = None) -> str:
        class=class="s">"s">class="s">"""开始一个新的 Trace。

        Args:
            name: Trace 名称（如 class="s">"user_query"）。
            session_id: 会话 ID。

        Returns:
            trace_id。
        class="s">"""
        trace_id = session_id or hashlib.md5(
            fclass="s">"{name}-{time.time()}".encode()
        ).hexdigest()[:16]

        span = TraceSpan(
            trace_id=trace_id,
            span_id=fclass="s">"span_{len(self.traces[trace_id])}",
            parent_span_id=None,
            name=name,
            start_time=time.time(),
        )
        self.traces[trace_id].append(span)
        self._active_spans[trace_id] = span
        return trace_id

    @contextmanager
    def span(self, trace_id: str, name: str, **attributes):
        class=class="s">"s">class="s">"""创建一个子 Span（上下文管理器）。

        用法:
          with tracer.span(tid, class="s">"llm_call", model=class="s">"gpt-4o"):
              result = llm.invoke(...)

        Args:
            trace_id: 父 Trace ID。
            name: Span 名称。
            **attributes: 附加属性。
        class="s">"""
        parent = self._active_spans.get(trace_id)

        span = TraceSpan(
            trace_id=trace_id,
            span_id=fclass="s">"span_{len(self.traces[trace_id])}",
            parent_span_id=parent.span_id if parent else None,
            name=name,
            start_time=time.time(),
            attributes=attributes,
        )
        self.traces[trace_id].append(span)
        if parent:
            parent.children.append(span)

        try:
            yield span
        except Exception as e:
            span.status = class="s">"error"
            span.attributes[class="s">"error"] = str(e)
            raise
        finally:
            span.end_time = time.time()
            span.attributes[class="s">"latency_ms"] = span.latency_ms

    def log_llm_call(self, trace_id: str, model: str,
                     input_tokens: int, output_tokens: int,
                     latency_ms: float, success: bool = True):
        class=class="s">"s">class="s">""class="s">"记录一次 LLM 调用（便捷方法）。"class="s">""
        with self.span(trace_id, class="s">"llm_call",
                       model=model,
                       input_tokens=input_tokens,
                       output_tokens=output_tokens,
                       success=success) as span:
            span.end_time = span.start_time + latency_ms / 1000

        self.metrics[class="s">"total_llm_calls"] += 1
        self.metrics[class="s">"total_input_tokens"] += input_tokens
        self.metrics[class="s">"total_output_tokens"] += output_tokens
        if not success:
            self.metrics[class="s">"failed_llm_calls"] += 1

    def log_tool_call(self, trace_id: str, tool_name: str,
                      input_args: dict, output_summary: str,
                      latency_ms: float, success: bool = True):
        class=class="s">"s">class="s">""class="s">"记录一次工具调用（便捷方法）。"class="s">""
        with self.span(trace_id, class="s">"tool_call",
                       tool_name=tool_name,
                       input_args=str(input_args)[:200],
                       output_summary=output_summary[:200],
                       success=success) as span:
            span.end_time = span.start_time + latency_ms / 1000

        self.metrics[class="s">"total_tool_calls"] += 1
        if not success:
            self.metrics[class="s">"failed_tool_calls"] += 1

    def get_trace(self, trace_id: str) -> Optional[dict]:
        class=class="s">"s">class="s">""class="s">"获取 Trace 的完整信息（用于调试和展示）。"class="s">""
        spans = self.traces.get(trace_id, [])
        if not spans:
            return None

        total_latency = sum(
            s.latency_ms for s in spans
            if s.end_time is not None
        )
        failed = sum(1 for s in spans if s.status == class="s">"error")

        return {
            class="s">"trace_id": trace_id,
            class="s">"span_count": len(spans),
            class="s">"total_latency_ms": round(total_latency, 1),
            class="s">"failed_spans": failed,
            class="s">"spans": [
                {
                    class="s">"name": s.name,
                    class="s">"latency_ms": round(s.latency_ms, 1),
                    class="s">"status": s.status,
                    class="s">"attributes": {
                        k: v for k, v in s.attributes.items()
                        if k in (class="s">"model", class="s">"tool_name", class="s">"input_tokens",
                                 class="s">"output_tokens", class="s">"success")
                    },
                    class="s">"children": len(s.children),
                }
                for s in spans
            ],
        }

    def get_metrics(self) -> dict:
        class=class="s">"s">class="s">""class="s">"获取聚合指标。"class="s">""
        return {
            class="s">"project": self.project_name,
            class="s">"traces": len(self.traces),
            class="s">"metrics": dict(self.metrics),
            class="s">"avg_tokens_per_call": (
                self.metrics[class="s">"total_input_tokens"] /
                max(self.metrics[class="s">"total_llm_calls"], 1)
            ),
            class="s">"tool_success_rate": (
                1 - self.metrics[class="s">"failed_tool_calls"] /
                max(self.metrics[class="s">"total_tool_calls"], 1)
            ) if self.metrics[class="s">"total_tool_calls"] > 0 else 0,
        }

    def print_trace_tree(self, trace_id: str, indent: int = 0):
        class=class="s">"s">class="s">""class="s">"打印 Trace 树形结构（直观展示执行流程）。"class="s">""
        spans = self.traces.get(trace_id, [])
        if not spans:
            print(class="s">"  无 Trace 数据")
            return

        class=class="s">"h"># 找到根 Span
        root = None
        for s in spans:
            if s.parent_span_id is None:
                root = s
                break

        if root is None:
            root = spans[0]

        def _print_span(span: TraceSpan, level: int):
            prefix = class="s">"  " * level + (class="s">"├─ " if level > 0 else class="s">"")
            status_icon = class="s">"✅" if span.status == class="s">"ok" else class="s">"❌"
            attrs = class="s">""
            if class="s">"model" in span.attributes:
                attrs += fclass="s">" [{span.attributes[class='s'>'model']}]"
            if class="s">"tool_name" in span.attributes:
                attrs += fclass="s">" [{span.attributes[class='s'>'tool_name']}]"
            print(fclass="s">"{prefix}{status_icon} {span.name}{attrs} "
                  fclass="s">"({span.latency_ms:.0f}ms)")

            for child in span.children:
                _print_span(child, level + 1)

        _print_span(root, 0)

    def print_dashboard(self):
        class=class="s">"s">class="s">""class="s">"打印监控仪表盘（模拟 Grafana 面板）。"class="s">""
        m = self.get_metrics()
        print(class="s">"=" * 55)
        print(class="s">"  📊 Agent 监控仪表盘")
        print(class="s">"=" * 55)
        print(fclass="s">"  项目: {m[class='s'>'project']}")
        print(fclass="s">"  Traces 总数: {m[class='s'>'traces']}")
        print(fclass="s">"  LLM 调用: {m[class='s'>'metrics'].get(class='s'>'total_llm_calls', 0)}")
        print(fclass="s">"  总输入 Tokens: {m[class='s'>'metrics'].get(class='s'>'total_input_tokens', 0):,}")
        print(fclass="s">"  总输出 Tokens: {m[class='s'>'metrics'].get(class='s'>'total_output_tokens', 0):,}")
        print(fclass="s">"  平均 Tokens/次: {m[class='s'>'avg_tokens_per_call']:.0f}")
        print(fclass="s">"  工具调用: {m[class='s'>'metrics'].get(class='s'>'total_tool_calls', 0)}")
        print(fclass="s">"  工具成功率: {m[class='s'>'tool_success_rate']:.0%}")
        status = class="s">"🟢 正常" if m[class='s'>'metrics'].get(class='s'>'failed_llm_calls', 0) == 0 else class="s">"🔴 异常"
        print(fclass="s">"  系统状态: {status}")
        print(class="s">"-" * 55)
